# XOR Transversal Torus Generator

This project explores the intersection of bitwise logic and topological geometry. It generates a 3D Torus structure where the surface geometry and probability density are dictated by a **XOR Transversal Stack**.

## How it Works
1. **Transversal Generation**: We compute a bitwise XOR matrix across a square coordinate plane. This creates a self-similar fractal-like pattern.
2. **Probability Stacking**: The XOR values are normalized into a dataset representing probability distributions ($P \in [0, 1]$).
3. **Toroidal Mapping**: We use the parametric equations of a torus, substituting the constant minor radius $r$ with our XOR-weighted probability stack.

## Replication
To replicate these results:
1. Install dependencies: `pip install numpy matplotlib pandas`
2. Run `python torus_engine.py`
3. The script will generate a 3D visualization and export the raw `xor_probability_set.csv`.

## Mathematical Foundation
The displacement $d$ at any point $(u, v)$ on the torus is defined by:
$$d = \text{norm}(XOR(u, v))$$
Where the final coordinates are:
$$X = (R + (r + d) \cos v) \cos u$$
$$Y = (R + (r + d) \cos v) \sin u$$
$$Z = (r + d) \sin v$$
